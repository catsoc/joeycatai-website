## 為什麼建這個網站？

每位工程師應該都有一個屬於自己的角落，可以自由地記錄想法、分享學習、展示作品。

這個網站是用 [Astro](https://astro.build) 建立的靜態網站，部署在 Cloudflare Pages 上，
既快速又免費——完全符合我的品味。

## 這裡會有什麼？

- **技術筆記**：踩坑紀錄、解決方案、工具使用心得
- **專案展示**：正在做的東西、已完成的作品
- **隨筆**：不那麼技術性的想法

## 技術棧

```
Astro 4.x        → 靜態網站框架
Tailwind CSS     → 樣式
MDX              → 可以在 Markdown 中用 JSX
Cloudflare Pages → 部署
```

就這樣，開始了。
